## Anti-Patterns Verdict

**LLM assessment:** Pass. No absolute-ban violations. Restrained color, authored feel, no kit-assembly markers. Detector found 1 false positive (broken-image rule on brand SVG fallback pattern).

## Overall Impression

Strong professional whiteboard with desktop-app sophistication. Alignment guides, full undo/redo with edge restoration, scenario branching, Figma-style shortcuts. Biggest opportunity: discoverability — power features are invisible to first-time users.

## What's Working

1. Keyboard shortcut parity with Figma/Funnelytics (Cmd+D/C/V/Z, Shift+drag, Space pan)
2. Alignment guides with visual-centre snapping (disc vs page mockup aware)
3. Undo/redo restores incident edges on step un-delete

## Priority Issues

### [P1] No sync status indicator on the board
Optimistic writes to Supabase have no UI feedback. Silent data loss risk on network failures.
Fix: Add syncStatus to FunnelBoardContext, render in header.

### [P1] No confirmation on multi-node delete
Backspace on 5+ selected nodes deletes instantly. Undo history capped at 30.
Fix: Gate onNodesDelete on confirm dialog when deleted.length > 1.
Command: /impeccable harden

### [P1] No onboarding tour or help reference
~10 discoverable features with no guidance. React-joyride already in project.

### [P2] Icon picker has 100+ icons with no search
Power users customize 50+ steps/week. Scrolling is 10-30s per step.
Fix: Add search input above icon grid, filter by slug substring.

### [P2] "Save as template" fires without confirmation
